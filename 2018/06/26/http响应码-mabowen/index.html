HTTP状态码

发表于 2018-06-26 | | 阅读次数:

  • 200s:请求被接受,并且服务器没有发生错误

  • 300s:重定向

  • 400s:客户端错误
    一般来说是指请求出现问题,这可能是浏览器原因,或者是使用得url有问题

  • 401s:未授权

  • 403s:禁止访问

  • 404s:不确定这个资源是否存在

  • 407s:需要代理服务器授权

  • 408s:超时

  • 409s:冲突

  • 410s:这是一种资源曾经存在的情况，但已被永久删除，并且不会返回。 如果您从网站上永久删除了一个页面，则应该使用 410，而不是 404。

  • 500s:服务器错误

  • 502s:网关错误

  • 503s:服务暂时不可用

  • 504s:网关超时
