PHP函数

发表于 2019-01-15 | | 阅读次数:

2019/1/23

  • array_combine(keys,values):array 创建一个数组,一个数组的值作为键名,另一个值作为其值

    1.keys 将被作为新数组的键,非法的值将会被转换为字符串类型
2.values 将作为array的值
3.return 合并的array,如果两个数组的单元数不同则返回false

  • array_count_values(array):array 统计数组中所有的值

    1.array 统计这个数组的值
2.return 一个关联数组,用array里数组中的值作为键名,该值在数组中出现的次数为值

  • array_diff_assoc(array,…array):array 带索引检查计数的差集

    1.return 一个数组,该数组包含第一个数组中不存在于任何其他数组中的所有值

  • array_diff_key(array,…array):array 使用键名比较计算数组的差集

    1.return 一个数组,该数组包含所有出现在第一个数组但是未出现在任何其他参数数组中的键的值

  • array_diff_uassoc 用用户提供的回调函数做索引检查来计算数组的差集

    2019/1/22

  • array_filter(array,callback,falg):array 用回调函数过滤数组中的单元

    1.array 要循环的数组
2.callback 使用的回调函数
3.flag 决定callback接收的参数形式
    1.ARRAY_FILTER_USE_KEY - callback接受键名作为的唯一参数
    2.ARRAY_FILTER_USE_BOTH - callback同时接受键名和键值
4.return 过滤后的数组

  • array_unshift(array,…):int 在数组开头插入一个或多个单元

    1.return array数组新的单元数目

  • array_shift (array):mixed 将数组开头的单元移除数组

    1.return 移出的值,如果array为空或者不是一个数组返回null

  • array_pop(array):mixed 弹出数组最后一个单元(出栈)

    1.array 需要弹出栈的数组
2.return array的最后一个值,如果array为空或者不是一个数组,返回null

  • array_push(array,value,…value):int 将一个或多个单元压入数组的末尾(入栈)

    1.array 输入的数组
2.value 压入array末尾的第一个值
3.return 处理之后数组的元素个数

2019/1/21

  • array_fill(start_index,num.value):array 用给定的值填充数组

    1.start_index 返回数组的第一个索引值
2.num 插入元素的数量,必须大于或等于0
3.value 用来填充的值
4.return 填充后的数组

  • array_chunk(array,size,preserve_keys):array 将一个数组分割成多个

    1.array 需要操作的数组
2.size 每个数组的单元数目
3.preserve_keys 为true可以保留原来的key,如果为false,重置索引,默认false
4.return 得到的数组是一个多维数组中的单元

  • array_slice(array,offset,length,preserve_keys):array 从数组中取出一段

1.array 输入的数组
2.offset 如果 offset 非负，则序列将从 array 中的此偏移量开始。如果 offset 为负，则序列将从 array 中距离末端这么远的地方开始
3.length 从offset开始的长度
4.preserve_keys 默认会重置数组的索引,为true不重置
5.return 其中的一段,如果offset参数大于array尺寸,就会返回空的array

  • array_diff(array,…array):array 计算数组的差集

    1.return 一个数组,该数组包含所有在第一个array,但是不在其他任何参数数组的值

  • array_intersect(array,…array):array计算数组的交集 

    1.return 一个数组,该数组包含了所有在第一个数组中也同时出现在其他参数数组的值

2019/1/20

  • strip_tags(string,allowable_tags):string 从字符串中除去html和php标记

    1.string 输入字符串
2.allowable_tags 使用可选的第二个参数指定不被去除的字符串列表
3.return 处理后的字符串

  • explode(delimter,string,limit):array 使用一个字符串分割另一个字符串

    1.delimiter 边界上的分隔字符
2.string 输入的字符串
3.limit 如果为正数,则返回数组最多包含limit个元素,而最后那个元素包含string的剩下部分
4.return 此函数返回由字符串组合的array,每个元素都是string的一个子串,如果delimiter为空字符串,返回false

  • implode(glue,pieces):string 将一个一维数组的值转化为字符串

    1.glue 默认为空的字符串
2.pieces 你想要转化的数组
3.return 字符串,其内容为由glue分割开的数组的值

  • substr(string,start,length):string 返回字符串的子串

    1.string 输入字符串,必须至少有一个字符
2.start 返回的字符串从string的start位置开始
3.length 从start开始的截取的长度
4.提取的子字符串,失败返回false

  • str_word_count(string,format,charlist):mixed 返回字符串中单词的使用情况

    1.string 字符串
2.format 指定函数的返回值
    0 - 返回单词的数量
    1 - 返回一个包含string中全部单词的数组
    2 - 返回关联数组,key是单词在string中出现的数值位置,value 是这个单词
3.charlist 附加的字符串,其中的字符串也被视为单词的一部分
4.return 返回一个数组或整型数,取决于format参数的选择

2019/1/19

  • abs(number):number 绝对值

  • str_pad(input,pad_length,pad_string,pad_type):string 使用另一个字符串填充字符串为指定长度

    1.input 输入的字符串
2.pad_length 如果这个的值是负数,小于或者等于输入字符串的长度,不会发生任何填充,并会返回input
3.pad_string 填充的字符串
4.pad_type 可选
5.return 填充的字符串

  • str_repeat(input,multiplier) 重复一个字符串

    1.input 待操作的字符串
2.multiplier 被重复的次数
3.return 重复后的字符串

  • str_split(string,split_length):array 将一个字符串转换为数组

    1.string 输入字符串  
2.split_length 每一段的长度
3.return 如果指定了split_length的参数,返回数组每个元素均为一个长度为split_length的字符块,否则每一个字符块为单个字符.如果split_length小于1,返回false,如果超过,整个字符串作为数组仅有一个的元素返回

  • strrev(string):string 反转字符串

2019/1/18

  • is_numeric(var):bool 检测变量是否为数字或者数字字符串

  • array_column(array,column_key,index_key):array 返回数组中指定的一列

    1.array 需要取出数组列的多维数组
2.column_key 需要返回值得列,如果是null,返回整个数组
3.index_key 返回数组得索引/键得列
4.return 从多维数组返回单列数组

  • array_search(needle,haystack,strict):mixed 在数组中搜索给定的值，如果成功则返回首个相应的键名

    1.needle 待搜索得值
2.haystack 待搜索得数组
3.strict 严格比较(===)
4.return 如果找到needle 则返回他的键,否则false

  • in_array(needle,haystack,strict):bool 检查数组中是否存在某一个值

    1.needle 待搜索得值
2.haystack 待搜索得数组
3.strict 严格比较(===)
4.return 找到返回true,失败返回false

  • array_unique(array,sort_flags):array 移除数组中重复得值

    1.array 输入得数组
2.sort_flags 可选参数sort_flags 可用于修改排序得行为
3.return 返回过滤后的数组

2019/1/17

  • array_keys(array,search_value,strict):array 返回array中的数字或者字符串的键名

    1.array 一个数组
2.search_value 如果指定了这个参数,只有包含这些值的键才会返回 不然会返回全部
3.strict 严格比较(===)
4.return 键名

  • array_values(array):array 返回数组中的值并给其建数字索引

  • array_merge(array,array,…):array 合并一个或多个数组

    1.array 合并的数组,可多个
ps.如果数组中有相同的字符串键名，则该键名后面的值覆盖前面的值；如果想让前面的值覆盖后面，则可以使用 + 号,如果是数字键名,后面的值不会覆盖原来的值,而是附加到后面

  • str_shuffle(string):string 随机打乱一个数组

    1.要打乱的字符串
2.return 打乱后的字符串

  • shuffle(array):bool 打乱数组

    1. array 待操作的数组
2. return 成功返回true 失败false

2019/1/16

  • iconv(in_charset,out_charset,string):string 将字符串string从in_charset转换编码到out_charset

    1.in_charset  必需,输入的字符集
2.out_charset 必需,输出的字符集
3.string 要转化的字符串
4.return 转换后的字符串,失败返回false

  • uniqid(prefix,more_entropy):string 

    1.prefix 前缀,为空返回长度13的字符串,more_entropy为true,返回23的字符串
2.more_entropy 为true,使得id更具有唯一性
3.return 一个带前缀,基于当前时间微秒生成得唯一ID

  • gettype(var):string 获取php变量的类型

  • settype(var,type):bool 设置php变量的类型

    1.var 要转化的变量
2.type 要变成的类型
3.return 成功返回true,失败false

  • getcwd():string 返回当前工作区目录

2019/1/15

  • strpos(string,find,start):int 查找字符串首次出现的位置 (对大小写敏感)

  • stripos(string,find,start):int 查找字符串首次出现的位置 (对大小写不敏感)

  • strrpos(string,find,start):int 查找字符串最后一次出现的位置 (对大小写敏感)

    1.string  必需,规定要搜索的字符串
2.find  必需,规定要查找的字符串  
3.start 可选,从何处开始搜索  
4.return 字符串在另一个字符串第一次出现的位置,没有找到返回false

  • strstr(string,search,before_search):string 搜索字符串在另一字符串中的第一次出现(对大小写敏感) 

    1.string 必需,规定要搜索的字符串
2.search 必需,规定所搜索的字符串
3.before_search 可选,默认false, 如果是true,则返回search参数之前第一次出现的字符串部分
4.return 字符串剩余部分,如果没有找到则返回false

  • strrchr(string,char):string 查找字符串在另一个字符串中最后一次出现的位置

    1.string 必需,规定要搜索的字符串
2.char 必需,规定要查找的字符串
3.return 从某个字符串在另一个字符串最后一次出现的位置到主字符串结尾的所有字符串,没有找到则返回false
